Formation SpringBoot

La formation se présentera sous la forme d’une alternance entre découverte de concepts de théoriques, annihilation d’idées reçues, questions / réponses, écriture de code, lecture de code et refactoring de code. L’outil de communication retenu pour cette formation est l’alternance entre l’oral, l’écrit, le partage d’écran si visio (audio/vidéo) et la manipulation d’un éditeur de code.

Contenu de la formation​

Jour 1​

Qu’est-ce qu’une API REST, Les différentes architectures applicatives, Spring VS Spring Boot, Spring Core, Spring Web, injection de dépendances et séparation des responsabilités, Tests unitaires en TDD, Intégration Continue avec gitlab/jenkins​

Jour 2​

Spring Data, Soumission et validation des données, Architecture Hexagonale, Tests d’intégration, Tests fonctionnels avec Cucumber​

Jour 3​

Spring Security, Gestion des rôles et permissions, Optimisation des performances de tests, Documentation Open API

Pour qui ?

Développeur en panique car il doit prochainement utiliser spring boot ​ Développeur qui a perdu le control de son appli spring-boot ​ Développeur qui a envie de rajouter une compétence dans sa boite à outil.

Acquis de la session​

Autonomie sur SpringBoot​ et qualité de code

Durée

3 jours

Déroulé de la session​

  • Apports théoriques sur le processus des exemples​
  • Démonstrations par le formateur​
  • Mises en pratique de ces apports théoriques​
  • Exercices en live coding interactif​
  • Refactoring de code pour inciter aux questionnements et échanges​
  • Application en condition réelle​

Intervenant

Coach Craft

Pré requis​

  • Bonne maîtrise de Java ou d’un langage similaire comme C#​
  • Bonnes connaissances en programmation orientée objet​
  • Capacité à écrire un simple test unitaire

Lieu

Chez vous ou dans nos locaux ou online​
